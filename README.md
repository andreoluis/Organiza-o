# Biblioteca Online 📚

Sistema fictício de gerenciamento de biblioteca digital, desenvolvido como projeto de estudo.

## Objetivo
Permitir que usuários cadastrem, pesquisem e reservem livros digitais de forma simples e organizada.

## Tecnologias previstas
- Python
- SQLite
- HTML/CSS

## Estrutura do projeto
- `docs/` — documentação e requisitos
- `src/` — código-fonte da aplicação
- `assets/` — imagens e recursos visuais

## Como executar
1. Clone o repositório
2. Acesse a pasta `src/`
3. Execute `python app.py`
